<h2>設定画面</h2>
<a href="https://github.com/CrossDarkrix/LiteDesktopStudio"><img src="https://raw.githubusercontent.com/CrossDarkrix/LiteDesktopStudio/refs/heads/main/Images/SettingWindow.png" width="300px" height="250px"></a>
<h3>★特徴</h3>
<ul>
	<li>設定画面へは右クリックを2回素早く押すことによって設定画面に入れるようになります。</li>
	<li>ウィジェットを1回左クリックし、Eキーを押すことで配置の編集画面には入れます。</li>
	<li>各ウィジェットは透明度が変更でき、アクセントカラーや背景の色も変更できます。</li>
	<li>簡単な操作で直感的な操作が可能です。</li>
	<li>上級者向けに簡単にですがHTML&CSSでウィジェットも作成可能。</li>
	<li>PySide6で実装されているのでソースコードを読める方なら自由に追加のカスタマイズが可能！</li>
</ul>

<h2>設定ファイルの場所</h2>
<ul>
	<li>Windowsならインストール場所直下に生成されます。</li>
	<li>Windows以外はスクリプトのある場所に配置されるようになっています。</li>
</ul>
